# PaintColorHQ Blog — Topical Plan (v1, 2026-06-01)

**Strategy:** brand-anchored discovery, to rebuild Google authority post-HCU and
reinforce the ~75% branded-lookup demand. Grounded in first-party GSC + Bing
demand data (see memory `reference_pchq-search-channel-intent-2026-05`) + a gap
analysis vs the existing 29 posts. Each post is written from the **23K-color DB**
(exact hex/LRV/undertone + cross-brand match / Delta-E data) + firecrawl/crawl4ai
research on primary sources — data-backed claims are what make it authoritative
and HCU-resistant.

**Cadence:** ~2 posts/week. **Author:** Philip Cameron (honest framing — no
fabricated credentials). Each post ships with a `gemini-3-pro-image` hero + a
matching Pinterest pin (auto-feeds the drip guide lane).

**The signature format (our moat):** wherever a post names colors, include the
DB's **cross-brand matches + coordinating colors + exact LRV** — content
competitors literally cannot reproduce. Lean on it.

## Priority / sequencing
Write in tier order (1 → 4). Tier 1 is the highest-ROI (fills empty brand gaps +
captures the biggest Bing terms + reinforces brand pages). The color deep-dive
(Tier 4 #17) is the **signature template** — once one is great, it repeats across
the top ~20 most-searched colors as an evergreen series, each uniquely data-backed.

## Parallel quick win (not blog)
Optimize `/brands/[slug]` pages to target **"[brand] color chart"** + **"all
[brand] colors"** (big branded-lookup queries they rank for but don't target).

## Per-post research (pipeline does this at draft time)
firecrawl/crawl4ai the SERP + primary brand sources for each title (validate
angle, harvest "people also ask", confirm facts), and pull the DB for exact
color data + cross-brand matches. No post ships on memory alone.
